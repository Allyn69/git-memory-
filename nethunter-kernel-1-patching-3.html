Patching the Kernel

We will continue from the Porting NetHunter page and work on the Neus 6P kernel as an example. The idea stays the same though.

 

Patching

By default, we apply wifi injection patches and patches that add wifi drivers. Kernel versions older than 4.x will require HID keyboard / mouse patches. We no longer need CD-ROM patches as all modern operating systems support being installed from USB storage devices, thus we can mount NetHunter as USB stick.
